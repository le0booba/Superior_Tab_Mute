   **Интеллектуальное управление звуком в Chrome**
   
   **[English](README.md)** | **Русский**

   Больше никакого звукового хаоса! Superior Tab Mute позволяет полностью контролировать звук в браузере, автоматически отключая фоновые вкладки по простым и понятным правилам. Создайте комфортную рабочую среду: сосредоточьтесь на важном, слушайте музыку без перебоев или выделите единственный источник звука. Интеллектуальные режимы отключения, мгновенное управление и удобный интерфейс — работайте в браузере без отвлекающих звуков.

## 🚀 Установка

1.  **Скачайте расширение**
    -   Загрузите [ПОСЛЕДНИЙ РЕЛИЗ](https://github.com/le0booba/Superior_Tab_Mute/releases/latest) с GitHub.
    -   Распакуйте скачанный архив.

2.  **Загрузите в Chrome**
    -   Откройте Chrome и перейдите на страницу `chrome://extensions/`.
    -   Включите **Режим разработчика** (переключатель справа вверху).
    -   Нажмите **Загрузить распакованное расширение** и выберите папку с расширением.

3.  **Начните использовать**
    -   Нажмите на иконку расширения в панели инструментов Chrome, чтобы открыть окно настроек.

## 🌟 Возможности

### 🎯 Умные режимы работы

-   **Режим активной вкладки**: Звук воспроизводится только на текущей активной вкладке. Идеально для работы, требующей концентрации.
-   **Режим первого звука**: Приоритет отдаётся первой вкладке, на которой появился звук. Отлично подходит для фонового прослушивания музыки или подкастов.
-   **Режим белого списка**: Вручную выберите единственную вкладку для воспроизведения звука. Точный контроль для сложных рабочих сценариев.
-   **Режим отключения новых вкладок**: Автоматически отключает звук на всех вновь открываемых вкладках. Ваша рабочая сессия останется тихой.

### ⚡ Быстрое управление

-   **Главный переключатель**: Включите или выключите расширение одним кликом или горячей клавишей (`Alt+S`).
-   **Глобальное отключение**: Мгновенно отключите звук сразу на всех вкладках (`Alt+A`).
-   **Установка источника звука**: Назначьте текущую вкладку источником звука в режиме первого звука (`Alt+W`).
-   **Показать все вкладки**: Просматривайте и выбирайте из всех доступных вкладок, а не только тех, где есть звук (в режимах первого звука и белого списка).
-   **Раскрываемые настройки**: Режимы первого звука и белого списка содержат скрываемые разделы с дополнительными настройками и списками вкладок. Интерфейс остаётся чистым и удобным. Нажмите кнопку раскрытия (▼) для доступа к расширенным настройкам.

### 🧠 Интеллектуальная автоматизация

-   **Запоминание последнего источника**: Если включить эту опцию для режимов «Первый звук» или «Белый список», расширение автоматически переключится на ранее звучавшую вкладку, когда текущий источник закроется. Вы слушаете музыку без перерывов — всё работает само, не требуя вашего участия.
-   **Настройки по умолчанию**: Задайте предпочитаемый режим и состояние «Отключить всё», которые будут применяться автоматически при каждом запуске Chrome. Отметьте любой режим или переключатель «Отключить всё» звёздочкой (★), чтобы сделать его настройкой по умолчанию.

<details>
<summary>🔧 Расширенная информация</summary>

#### Базовая архитектура:
1. **Постоянные синхронизируемые настройки**: Основные настройки синхронизируются между устройствами через `chrome.storage.sync`, обеспечивая единообразный опыт работы. Настройки включают выбор режима, состояние расширения, статус отключения звука и предпочтение запоминания последнего источника.
2. **Безопасная обработка вкладок**: Автоматически исключает системные страницы Chrome (`chrome://`) и страницы расширений (`chrome-extension://`) во избежание конфликтов с помощью проверки `isManageableTab()`.
3. **Восстановление после ошибок**: Корректно обрабатывает закрытые вкладки через обёртки `safeGetTab()`, `safeUpdateTab()` и `safeQueryTabs()`, которые перехватывают ошибки и логируют их, не нарушая работу расширения.
4. **Многоуровневая система хранения**: 
   - `chrome.storage.sync`: Пользовательские настройки (режим, параметры, значения по умолчанию), синхронизируемые между устройствами
   - `chrome.storage.session`: Временные ID вкладок, история источников и состояния раскрытия секций, удаляемые при перезапуске браузера
   - `chrome.storage.local`: Локальные настройки интерфейса (язык, состояния чекбоксов) для конкретного устройства
5. **Поддержка нескольких языков**: Встроенная английская и русская локализация с плавным переключением между языками. Языковые настройки хранятся локально, тексты интерфейса динамически загружаются из файлов `_locales/` JSON при инициализации всплывающего окна.

**Оптимизация производительности:**
1. **Отложенная обработка событий (debouncing)**: Основные операции отключения звука, такие как `applyMutingRules()`, выполняются с задержкой 150 мс для предотвращения избыточных вызовов при быстром переключении и обновлении вкладок, снижая нагрузку на процессор во время интенсивной работы в браузере.
2. **Параллельное выполнение**: Использование `Promise.all()` для одновременного выполнения независимых операций при получении настроек (`refreshCache()`), инициализации (`handleStartup()`, `handleInstall()`) и массовой обработке вкладок, что существенно сокращает общую задержку.
3. **Кешированное управление настройками**: Реализация кеша настроек в памяти (`cachedSettings`) с быстрым доступом через `getSettings()` и защитой от дублирования через `settingsPromise`, объединяющей конкурентные вызовы обновления в одно обращение к хранилищу. Минимизирует избыточные обращения к API в событийно-насыщенных сценариях.
4. **Избирательная обработка вкладок**: Предварительная фильтрация управляемых вкладок с помощью `isManageableTab()` перед любыми операциями — системные страницы исключаются на входе, что сокращает количество вызовов Chrome API во всех операциях управления звуком.
5. **Проверка текущего состояния перед обновлением**: Перед каждым вызовом `chrome.tabs.update()` текущее состояние mute вкладки сравнивается с целевым — вкладки, уже находящиеся в нужном состоянии, пропускаются. Фильтрация реализована в `setTabsMuted()` через проверку `tab.mutedInfo?.muted`, что устраняет значительную долю лишних вызовов Chrome API при каждом применении правил.

</details>

## 📖 Как пользоваться

1.  **Откройте всплывающее окно**
    -   Нажмите на иконку Superior Tab Mute в панели инструментов Chrome для доступа к основным настройкам.

2.  **Выберите режим работы**
    -   Выберите нужную стратегию отключения звука: **Активная вкладка**, **Первый звук**, **Белый список** или **Отключение новых вкладок** с помощью переключателей.
    -   Для режимов «Первый звук» и «Белый список» нажмите кнопку раскрытия (▼) внизу всплывающего окна, чтобы увидеть дополнительные настройки и список вкладок. Нажмите ещё раз, чтобы показать больше опций.
    -   Используйте список вкладок для выбора источника звука. Можете включить опцию «Показать все вкладки» для расширенного выбора.
    -   При работе в режимах «Первый звук» или «Белый список» установите флажок **«Запоминание последнего источника»**, чтобы включить автоматическое переключение источника. Расширение будет плавно переключаться на ранее звучавшие вкладки, если текущий источник закроется.

3.  **Установите настройки по умолчанию**
    -   Нажмите на иконку звезды (☆) рядом с любым режимом или переключателем «Отключить всё», чтобы сделать его настройкой по умолчанию.
    -   Закрашенная звёздочка (★) показывает текущую настройку по умолчанию.
    -   Эти настройки автоматически применяются при каждом запуске Chrome, обеспечивая стабильную работу между сессиями.

4.  **Используйте горячие клавиши**
    -   `Alt+S`: Включить/выключить расширение
    -   `Alt+A`: Переключить отключение звука на всех вкладках
    -   `Alt+W`: Установить текущую вкладку источником звука (режим первого звука)
    -   Наведите курсор на элементы управления, чтобы увидеть назначенные им горячие клавиши
    -   Настроить горячие клавиши можно на странице `chrome://extensions/shortcuts`

## 🔒 Разрешения и конфиденциальность

### Гарантия конфиденциальности

-   **Никакого сбора данных**: Расширение не собирает, не хранит и не передаёт никакие личные данные.
-   **Никакой аналитики**: Не содержит скриптов отслеживания и не связывается с внешними серверами.
-   **Локальная работа**: Вся функциональность работает исключительно локально в вашем браузере.

<details>
<summary>Используемые разрешения</summary>

-   **`tabs`**: Необходимо для определения вкладок, воспроизводящих звук, чтения их заголовков и значков для интерфейса, а также применения правил отключения звука.
-   **`storage`**: Используется для сохранения пользовательских настроек локально и между устройствами, чтобы обеспечить единообразный опыт работы между сессиями браузера.

</details>

<details>
<summary>Синхронизируемые настройки (chrome.storage.sync)</summary>

-   Настройки, которые хранятся в вашем аккаунте Google и синхронизируются между устройствами, на которых вы вошли в Chrome.
-   **`isExtensionEnabled`** (true/false): Управляет активностью расширения.
-   **`mode`** ('active', 'first-sound', 'whitelist', 'mute-new'): Определяет активный режим отключения звука.
-   **`isAllMuted`** (true/false): Переключает глобальное состояние отключения звука для всех вкладок.
-   **`rememberLastTab`** (true/false): Запоминает настройку функции «Запоминание последнего источника».
-   **`defaultMode`** (строка или null): Хранит предпочитаемый режим по умолчанию для применения при запуске.
-   **`defaultMuteAll`** (true/false): Определяет, должно ли «Отключить всё» быть включено по умолчанию при запуске.
-   *Назначение*: Обеспечивает единообразие основных настроек на всех ваших устройствах.

</details>

<details>
<summary>Сессионное хранилище (chrome.storage.session)</summary>

-   Временные настройки, которые удаляются при закрытии браузера.
-   **`firstAudibleTabId`** (ID вкладки): Отслеживает назначенную вкладку-источник звука в режиме «Первый звук».
-   **`whitelistedTabId`** (ID вкладки): Отслеживает выбранную пользователем вкладку в режиме «Белый список».
-   **`expansionStates`** (объект): Запоминает, какие раскрываемые секции открыты для каждого режима в текущей сессии.
-   **`muteNewInitialTabIds`** (массив): Отслеживает вкладки, существовавшие при активации режима «Отключение новых вкладок», используется для управления включением звука.
-   **`firstSoundSourceHistory`** (массив): Хранит ID до 3 последних активных источников звука в режиме «Первый звук»; используется функцией «Запоминание последнего источника» для переключения на активную альтернативу при закрытии текущего источника.
-   **`whitelistSourceHistory`** (массив): Хранит ID до 3 последних активных источников звука в режиме «Белый список»; используется функцией «Запоминание последнего источника» для переключения на активную альтернативу при закрытии текущего источника.
-   *Назначение*: ID вкладок уникальны для каждой сессии браузера и становятся недействительными между устройствами или после перезапуска, поэтому сессионное хранилище — оптимальный выбор.

</details>

<details>
<summary>Локальное хранилище (chrome.storage.local)</summary>

-   Настройки, которые сохраняются на устройстве, но не синхронизируются между аккаунтами.
-   **`stm_lang`** ('en'/'ru'): Запоминает выбранный язык интерфейса.
-   **`showAllTabsFirstSound`** (true/false): Запоминает состояние чекбокса «Показать все вкладки» для режима первого звука.
-   **`showAllTabsWhitelist`** (true/false): Запоминает состояние чекбокса «Показать все вкладки» для режима белого списка.
-   *Назначение*: Позволяет иметь разные настройки интерфейса на разных устройствах, например, различающиеся на рабочем и домашнем компьютерах.

</details>
